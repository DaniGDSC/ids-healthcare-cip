# Data Flow

1. Raw data ingested from data/raw/CSE-CIC-IDS2018
2. Phase 1 preprocesses → data/processed
3. Phase 2 feature selection → data/features
4. Phase 3 autoencoder → data/latent and models/phase3
5. Phase 4 clustering → results/phase4
6. Phase 5 classification → results/phase5 and models/phase5
7. Reports aggregated in results/reports
